## 📖 Sobre

Repositório com meus estudos e práticas durante um curso de **PHP** com foco em backend development e integração com **SQL**.

O objetivo final é aprender PHP e SQL de forma sólida para construir aplicações completas com CRUD.

## 🎯 Objetivos

- [x] Aprender fundamentos de PHP (sintaxe, lógica, funções e estrutura de código)
- [ ] Evoluir para integração com SQL (consultas, manipulação e modelagem de dados)
- [ ] Construir base sólida para desenvolvimento web backend
- [ ] Desenvolver um sistema CRUD completo como projeto final
